Enhancements made to this website. Part 2

Automatic Total Price Updater

The final price updater automatically calculates the price of the selected phone, multiplied by the quantity selected. It will also add any selected extra on top of that and will dynamically change whenever one of these elements is altered. The way this works is using javascript in the part2.js I am able to detect when each field is changed so for all those events I can run essentially the same code and that is how the total price always is updated and accurate.

Dynamic Extras List

The Dynamic Extras list was one of the harder elements to put together on my website. Depending on how many extras are selected the payments page is updated to show a list of selected extras which will change in size depending on what element is selected and will never display a blank entry when less than the maximum elements are chosen. The way this is accomplished is within the part2.js the selected extras are pushed into a blank array which is then cycled through using a for loop. The way the blank entries are dealt with is to delete the labels which are redundant creating the dynamic sizing of the list.
